# BlueNotes
一个C#编写的简单易用、功能单一的桌面便签小软件。使用Visual Studio 2026编译通过。  
软件本身不大，但是依赖微软运行时库：windowsdesktop-runtime-8.0.28-win-x64.exe 下载地址为：
https://aka.ms/dotnet-core-applaunch?missing_runtime=true&arch=x64&rid=win-x64&os=win10&apphost_version=8.0.28&gui=true  
若要打包为一个独立运行的包，体积约为150M。
软件功能：
1. 不用安装，解压后直接运行 BlueNotes.exe，因为设置了开机自动运行，360杀毒软件可能会报警，点击确认即可。开机运行功能可以在软件中关闭。  
2. 软件界面分为3个部分：  
（1）顶部菜单条功能为：根据输入内容查找便签、添加便签、置顶、最小化及系统菜单。  
（2）左边分类列表按键：  
   可以选中某个分类，再点击菜单条的+号来添加新的便签；  
   可以点击“+新建”按键来添加新的分类便签按键，分类按键底部有数字提示该分类所拥有的便签数量；  
   在每个分类按键上点击右键可以批量删除该分类的全部便签；  
   在新建的分类按键上点击右键可以删除该分类按键及所属的全部便签。  
   所有删除功能都不能恢复。  
（3）右边的便签列表：显示某个分类下的全部便签。  
   点击菜单条的+号添加新便签时，可以输入便签的标题，如果不输入，模式显示该便签建立时的时间。  
   点击便签上笔的图标可以修改便签内容及颜色等；  
   点击便签上垃圾桶图标可以删除便签，便签删除不可逆；  
   便签里面，顶部右方有一个时钟的图标，可以设置提醒，请严格按照格式输入；  
   当系统时间到达提醒时间时，软件会弹出对话框并发出叮的声音进行提醒；  
   当系统时间超过提醒时间1分钟后，提醒失效。  
   设置了提醒后，便签底部会有设置的时间显示；回到主视图后便签上也会显示时钟的图标。
